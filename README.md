# LLM API Cost Comparison Simulation

This example demonstrates how to compare the hypothetical API costs of different Large Language Models (LLMs) based on their pricing structures and estimated token usage. It simulates a text generation scenario and calculates the total cost for each model, illustrating the core concept of evaluating LLMs for a project.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
